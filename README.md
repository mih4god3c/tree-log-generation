# Log Generator

A procedural 3D tree log generator built with Three.js.

## Running

```bash
npm install
npm run app
```

## Parameters

- **Shape**: length, diameter, taper, eccentricity
- **Curvature**: amount and direction
- **Surface**: roughness, roughness scale
- **Gnarls**: count, size, strength, distribution
- **Bark**: type (birch, oak, pine, willow), flat shading, textured
- **Cross-section**: ring count, heartwood ratio, colors, pith offset, ring wobble, cracks
